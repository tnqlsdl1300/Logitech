# Logitech
_주변기기 판매 쇼핑몰_

## 소개

- 마우스, 키보드, 헤드셋 등의 주변기기를  판매하는 주변기기 판매 쇼핑몰 입니다.
- 개발기간: 2020.10.28 ~ 2020.11.30 (약 4주)
- 개발인원: 5명

## 기술스택

> Back-end/Server

- Java
- Jsp
- Java Servlet
- Apache Tomcat v8.5
- OracleDB

> Front-end

- Javascript
- JQuery
- HTML 5 & CSS3
- Bootstrap 3

> Devops

- Eclipse
- Git
- Github
